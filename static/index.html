什么是官方操练场兼容模式？

当官方 API 节点过载或常规接口频繁失败时，此模式会使用官网登录后的会话 Cookie 调用操练场接口，兼容到当前生成流程。

使用限制

图片质量和稳定性不可控；目前大概率只适配 gpt-image-2-flatfee 模型，以及 auto 分组 Key。

参数变化

激活官方操练场兼容模式

        
登录 APIQIK 官网账号，获取本次浏览器会话可用的官网 Cookie。

      

      

        
工作原理：本地服务会用下面的账号密码向 APIQIK 官网登录，然后把获得的会话 Cookie 用于操练场接口。

        
保存规则：Cookie 只绑定当前浏览器的 session_id。关闭兼容模式只会停用该通道，不会清除会话；服务端会话丢失或 Cookie 过期后才需要重新登录。

        
使用前确认：该通道用于官方 API 过载时兜底，质量不可控；大概率只支持 gpt-image-2-flatfee 和 auto 分组
          Key。
